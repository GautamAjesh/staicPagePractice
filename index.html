Position 1: Junior Software Developer

Position Description Reference: JSD12

Position Title: Junior Software Developer

Brief Description: We are looking for a motivated Junior Software Developer to join our development team and assist in building innovative software solutions.

Salary Range: $50,000 - $60,000 per year

Reporting to: Senior Software Developer

Key Responsibilities:

  • Participate in the design, development, and testing of software applications
  • Collaborate with team members to implement new features and enhancements
  • Debug and troubleshoot software issues
  • Contribute to code reviews and continuous improvement

Required Qualifications:

  1. Essential: Bachelor's degree in Computer Science or related field
  2. Essential: Proficiency in at least one programming language (e.g., Java, Python, C++)
  3. Preferable: Familiarity with web development technologies (HTML, CSS, JavaScript)
  4. Preferable: Experience with version control systems (e.g., Git)

Position 2: IT Project Manager

Position Description Reference: ITP12

Position Title: IT Project Manager

Brief Description: We are seeking an experienced IT Project Manager to oversee the planning, execution, and delivery of various IT projects within our organization.

Salary Range: $80,000 - $100,000 per year

Reporting to: Director of IT

Key Responsibilities:

  • Define project scope, goals, and deliverables in collaboration with stakeholders
  • Create and manage project plans, timelines, and budgets
  • Coordinate cross-functional teams and assign tasks
  • Monitor project progress and address any issues that arise
  • Ensure projects are delivered on time and within budget

Required Qualifications:

  1. Essential: Bachelor's degree in IT or related field
  2. Essential: 3+ years of project management experience
  3. Essential: Strong communication and leadership skills
  4. Preferable: Project Management Professional (PMP) certification

Position 3: Virtual Reality Developer

Position Description Reference: VRD12

Brief Description: Join our innovative team as a Virtual Reality Developer, where you'll be responsible for creating captivating and immersive virtual reality experiences.

Salary Range: $70,000 - $90,000 per year

Reporting to: Lead Developer

Key Responsibilities:

  • Collaborate with artists, designers, and other developers to bring ideas to life

  • Debug and troubleshoot technical issues

Required Qualifications:

  1. Essential: Bachelor's degree in Computer Science or related field
  2. Essential: Proficiency in Unity3D or Unreal Engine
  3. Essential: Experience with 3D graphics and VR development
  4. Preferable: Familiarity with C# or C++ programming languages
